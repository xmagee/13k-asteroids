# 13k-asteroids
Entry for js13kgames 2021

## run this 
* `$ git clone https://github.com/xmagee/13k-asteroids`
* `$ cd 13k-asteroids && npm install`
* `$ npm run dev`

## build this 
* `$ git clone https://github.com/xmagee/13k-asteroids`
* `$ cd 13k-asteroids && npm install`
* `$ npm run build`
* minified build will be in `build/` folder
